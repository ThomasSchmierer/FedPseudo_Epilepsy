The files here contain information for the execution of the FLPseudo_Epilepsy_Pred/Detect algorithm.

This is a federated learning algorithm for the model development of non-IID medical signal data. Specifically developed for EEG based seizure analysis.

This work supports the methods published by Thomas Schmierer at UniSQ in "EEG based federated learning for prediction and detection of Epilesy seizures"
